CHAPTER 03

교육

교육 완료 등록, 숙제 출제, 교육 실적
Education · 3.1

✅ 교육 완료 등록

사이드바 → "교육 완료". 인턴에게 어떤 주차의 교육을 완료했는지 기록.
1
교육 날짜 선택
오늘이 기본값
2
교육 레벨 선택
BASIC ~ GOLD-3 중 선택. 레벨 선택 시 해당 레벨 인턴만 자동 필터링
3
교육 주차 선택
1~16주차 중 선택. 순서 무관 (예: 2주차 다음 14주차도 OK)
4
수강생 체크
해당 레벨의 인턴 목록에서 교육 받은 인턴들 다중 선택
5
이번주 숙제 입력 (선택)
입력 시 자동으로 인턴들에게 숙제 등록
6
다음 교육 담당자 선택 (선택)
7
"교육 완료 등록" → 완료
자동으로 인턴들의 진도(주차 grid) 업데이트 + 이전 미완료 숙제 자동 완료
✓
레벨 → 주차 → 인턴 순서로 입력하면 자동 필터링되어 빠릅니다. 한 번에 여러 인턴 동시 등록 가능.
i
교육 횟수 = 본사 교육비 산정 기준. 1회당 ₩60,000이 자동 계산됩니다 (인원 무관).
!
실수로 잘못 등록한 경우, 본사 관리자에게 요청해서 "아카데미 관리"에서 수정해야 합니다.

📝 숙제 출제

사이드바 → "숙제 출제". 특정 인턴에게 별도 숙제 부여 가능 (교육 완료 등록과 별개).
1
인턴 선택
드롭다운에서 본인 지점/브랜드 인턴 중 선택
2
마감일 설정 (선택)
언제까지 완료해야 하는지 명시
3
숙제 내용 입력
예: "롤브러쉬 30회 두 손 연습", "샴푸 5명 완료"
4
출제 → 인턴에게 알림
인턴 대시보드에 미완료 숙제로 표시됨
자동 완료 처리
✓ 인턴이 직접 "완료" 버튼 클릭 → 완료 표시

✓ 다음 교육 완료 등록 시 → 해당 인턴의 이전 미완료 숙제 자동 완료

✓ 미완료 숙제는 인턴 대시보드 상단에 주황 알림으로 강조
i
교육 완료 등록 화면에서도 "이번주 숙제"를 같이 입력 가능 → 별도 출제 안 해도 됨

발주 / 근태

발주하기, 휴무 신청
Order & Attendance · 4.1

📦 발주하기 / 🌴 휴무 신청

디자이너 본인의 근태 처리는 휴무 신청 위주. 발주 권한 ON일 때만 발주 가능.
📦 발주하기 (권한 시)
1
사이드바 → "발주하기"
발주 권한 OFF면 메뉴 자체가 안 보임
2
카테고리 → 제품 선택
검색창으로 빠른 검색 가능
3
수량 입력 → 카트 추가
4
"발주 제출" → 본사 처리 대기
🌴 휴무 신청
1
사이드바 → "휴무 신청"
2
날짜 + 사유 입력
개인 사정, 병원, 행사 등
3
신청 → 본사 승인
디자이너는 인턴과 달리 "휴무"로 분류 (연차 차감 X)
i
대표 (민경/세운/김민경/김세운)는 사이드바 하단에 🚗 외근 버튼이 추가로 표시됩니다.

FAQ · 5.1

❓ 자주 묻는 질문

Q.담당 인턴이 사라졌어요
본사에서 데이터가 초기화됐을 가능성. 본사 관리자에게 "아카데미 관리"에서 자동 복구 또는 수동 재배정을 요청해주세요.
Q.잘못된 주차로 교육 완료 등록했어요
본사 관리자에게 요청 → "아카데미 관리" → 인턴 카드 클릭 → 주차 그리드에서 잘못된 주차 토글로 해제하고 정확한 주차 체크.
Q.인턴이 외부 아카데미를 다녀왔다는데 진도가 안 늘었어요
인턴이 본인 사이드바 → "외부 수강"에서 직접 등록해야 진도 반영됩니다. 안내해주세요.
Q.교육 완료 화면에 인턴이 안 보여요
레벨 선택 시 해당 레벨의 인턴만 표시됩니다. 다른 레벨 인턴이면 레벨을 다시 선택하세요. 인턴이 본인 지점이 아니거나 퇴사 처리됐을 수도 있습니다.
Q.휴무 신청을 인턴 휴가로 잘못 신청했어요
디자이너는 사이드바에 "휴무 신청"만 있고 "휴가 신청"은 없습니다. 잘못 표시되는 거면 본사 관리자에게 직급 확인 요청.
