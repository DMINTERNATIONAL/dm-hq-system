발주하기 (권한 시), 휴무 신청, FAQ
Order & Attendance · 3.1

📦 발주하기 / 🌴 휴무 신청

발주는 권한이 ON일 때만, 휴무 신청은 누구나 가능.
📦 발주하기 (권한 시)
1
사이드바 → "발주하기"
2
카테고리 → 제품 → 수량
검색창으로 빠르게 찾기 가능
3
"발주 제출" → 본사 처리 대기
4
"발주 내역"에서 진행 상태 확인
!
발주 권한 OFF면 메뉴 자체가 안 보임. 본사 관리자에게 권한 요청.
🌴 휴무 신청
1
사이드바 → "휴무 신청"
2
날짜 + 사유 입력
3
신청 → 본사 승인 대기
i
카운터는 "휴무"로 분류 (인턴의 "휴가/연차"와 다름). 잔여 일수 차감 X.

FAQ · 3.2

❓ 자주 묻는 질문

Q.발주하기 메뉴가 안 보여요
발주 권한이 OFF 상태입니다. 본사 관리자에게 권한 ON 요청. 직원 관리에서 본인 카드의 "발주OFF" 뱃지 확인 가능.
Q.다른 지점 인턴/발주가 안 보여요
카운터는 본인 지점만 볼 수 있습니다. 전 지점 통합 조회는 본사 관리자 권한이 필요합니다.
Q.발주 신청 후 취소하고 싶어요
본사 관리자에게 직접 요청해주세요. 본사가 "발주 관리"에서 해당 발주를 "발주 취소" 상태로 변경합니다.
Q.제품이 잘못 왔어요. 환불 요청하려면?
발주 내역 → 해당 발주 (배송완료 상태) → "환불 요청" 버튼 → 사유 입력. 본사가 환불 처리 후 "환불완료"로 변경.
Q.새 기능이 안 보여요
옛 PWA 캐시 문제. 홈 화면 DM 아이콘 삭제 → Safari로 다시 접속 → 홈 화면에 추가. 다음부터 자동 업데이트.
