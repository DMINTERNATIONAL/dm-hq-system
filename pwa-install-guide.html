홈 화면에 있는 DM 아이콘이 새 기능을 안 보여줄 때, 아래 순서대로 따라 해 주세요.
!
왜 재설치가 필요한가요?
이전 버전 앱이 옛날 화면을 캐시(저장)해두고 있어서 새로 추가된 기능이 안 보일 수 있습니다. 한 번만 아이콘을 다시 깔면 그 다음부터는 자동으로 최신 버전이 적용됩니다.
📲 QR 빠른 접속
아이폰 카메라로 스캔
QR을 스캔하면 Safari가 자동으로 열리고 시스템 페이지로 이동합니다. 그 후 아래 4·5단계(공유 → 홈 화면에 추가)만 따라하면 됩니다.
dminternational.github.io/dm-hq-system
설치 / 재설치 단계
🆕 신규 직원이라면: 위 QR을 스캔 → Safari가 열림 → 바로 4단계로 점프하면 됩니다.
🔄 기존 직원 재설치: 1단계부터 차례로 진행해주세요.
1
홈 화면의 기존 DM 아이콘 삭제
홈 화면에 있는 DM 아이콘을 꾹 길게 누르고 떼지 마세요.
① 길게 누르기 → 메뉴가 뜨면 "앱 삭제" 또는 "즐겨찾기 제거" 선택
② 확인 창에서 "홈 화면에서 제거" 누르기
※ 데이터는 사라지지 않습니다. 로그인 정보도 유지됩니다.
2
Safari 브라우저 열기
반드시 🧭 Safari에서 열어주세요. Chrome / 카카오톡 / 인스타 내장 브라우저에서는 홈 화면 추가가 안 됩니다.
3
주소창에 시스템 URL 입력
Safari 상단(또는 하단) 주소창에 아래 주소를 정확히 입력하세요.
https://dminternational.github.io/dm-hq-system/
💡 Tip: 이 PDF를 iPhone에서 열고 있다면, 위 주소를 길게 눌러 복사 → Safari 주소창에 붙여넣기
4
공유 버튼 → "홈 화면에 추가" 선택
화면 하단(또는 상단)의 ⬆ 공유 버튼을 누르세요.
① 공유 메뉴가 열리면 아래로 스크롤
② "홈 화면에 추가" 항목 찾아서 누르기
③ 이름 확인 (DM INTERNATIONAL) → 우측 상단 "추가" 누르기
5
홈 화면의 새 DM 아이콘으로 접속
홈 화면으로 돌아가면 새로운 DM 아이콘이 보입니다. 탭해서 접속하면 최신 버전이 표시됩니다.
🎉 다음부터는 자동 업데이트가 적용되어 이런 작업을 다시 하지 않아도 됩니다.
✓
재설치 완료!
기존 로그인 정보는 그대로 유지됩니다. 평소처럼 사용해 주시면 됩니다.
