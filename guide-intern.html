📱 앱 설치하기

아이폰 Safari 또는 안드로이드 Chrome으로 시스템 주소에 접속한 후 홈 화면에 추가하세요.
1
QR 스캔 또는 주소 입력
신규 직원은 본사에서 받은 QR 코드를 폰 카메라로 스캔합니다.

2
자동 안내 배너 확인
접속 후 "📱 앱처럼 빠르게 사용하기" 배너가 뜨면 "설치" 버튼을 누릅니다.

홈 화면에 추가 (iOS)
아이폰: 공유 버튼 (⬆) → "홈 화면에 추가" → "추가"
4
홈 화면 아이콘으로 접속
홈 화면에 생긴 DM 아이콘을 탭해서 접속합니다.

🔑 로그인 / 첫 가입

전화번호와 비밀번호로 접속합니다. 신규 가입은 본사 관리자 승인 후 사용 가능합니다.
기존 계정 로그인
1
"로그인" 버튼 클릭
시작 화면에서 로그인 카드를 탭합니다.
2
전화번호 + 비밀번호 입력
- 없이 숫자만 입력 (예: 01012345678)
3
자동 로그인 체크 (선택)
매번 입력하기 번거로우면 체크박스 활성화
신규 가입 신청
1
"가입 신청" 카드 클릭
2
정보 입력
실명, 활동명(2글자), 생년월일, 전화번호, 비번, 브랜드(DAY:MEAN), 지점, 직급(인턴), 현재 레벨, 마지막 수강 주차
3
본사 관리자 승인 대기
승인 후 로그인 가능. 보통 당일~익일 처리
i
비밀번호 잊었으면 "비밀번호 재설정"에서 이름·전화번호·생년월일 입력 → 본사 관리자 처리

📦 제품 / 비품 발주하기

사이드바 → "발주하기". 카테고리에서 제품 선택 → 수량 입력 → 발주 제출.
1
카테고리 선택
샴푸, 트리트먼트, 펌제, 비품 등 분류 선택
2
제품 검색 또는 목록에서 선택

3
수량 입력 → 카트 추가
+ / − 버튼으로 수량 조절
4
카트 확인 후 발주 제출
최종 확인 → "발주 제출" → 본사 관리자에게 전송
!
발주 권한이 OFF인 경우 사이드바에 "발주하기" 메뉴 자체가 안 보입니다. 본사 관리자에게 권한 요청 필요.

탭으로 필터

Q.비밀번호를 잊었어요
로그인 화면 → "비밀번호 재설정" → 이름·전화번호·생년월일 입력 → 본사 관리자가 새 비번 발급. 또는 본사 관리자에게 직접 연락이 더 빠릅니다.
Q.새 기능이 안 보여요
옛 버전 캐시 문제입니다. PWA를 한 번 재설치 (홈 화면 아이콘 삭제 → Safari로 다시 접속 → 홈 화면 추가). 다음부턴 자동 업데이트.
Q.주차 표시가 이상해요 (15주차인데 7/16으로 나옴)
"수강한 주차 / 총 주차" 형식입니다. 7/16은 16주차 중 7주차를 받았다는 뜻 (반드시 1~7번이 아닐 수도 있음 — "내 진도"에서 그리드 확인).
Q.발주하기 메뉴가 안 보여요
발주 권한이 OFF 상태입니다. 본사 관리자에게 권한 요청을 해주세요.
Q.휴게 60분 초과하면 어떻게 되나요?
자동 정지되지 않습니다. 본사 통계에 "휴게초과" 회수가 카운트됩니다. 직원 관리에서 이번달 초과 회수 표시.
